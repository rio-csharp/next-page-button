
# Next Page Button

English | [中文](https://github.com/rio-csharp/next-page-button/blob/main/README_zh_CN.md)

> Navigate your documents sequentially like an eBook. Seamlessly moving between pages in the order of your document tree.

## ✨ Features

- **Flexible Layout**: Choose between **Bottom Static** (classic) or **Side Floating** (modern) page buttons via settings.
- **Smart Navigation**: Automatically follows your document tree order, respecting notebook isolation and drag-and-drop reordering.
- **Interactive UI**: Real-time position indicator (e.g., `5 / 20`) and theme-adaptive styling.
- **Seamless Updates**: Instantly reacts to doc creation, deletion, or renaming.
- **Mobile Optimized**: Intelligent auto-hide functionality when the virtual keyboard is active (Android/iOS).
- **Internationalization**: Full support for English and Chinese, with immediate UI updates upon switch.

## 🚀 Installation

1. Open **Settings** -> **Marketplace** -> **Plugins**.
2. Search for `Next Page Button`.
3. Click **Download** and **Enable**.

## ⚙️ Configuration

Go to Plugin Settings to customize:
- **Layout Mode**: Toggle between Bottom or Side navigation.
- **Margins**: Adjust spacing for the bottom layout.
- **Language**: Manual override or auto-detect system language.

## 📱 Mobile Experience

Optimized for small screens:
- Floating buttons auto-hide when editing to prevent obstruction.
- Smooth transitions using native SiYuan mobile APIs.
