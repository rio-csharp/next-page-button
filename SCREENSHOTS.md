# 截图指南 / Screenshot Guide

## 需要的图片 / Required Images

### 1. icon.png
- **用途**：插件图标，显示在插件列表中
- **尺寸**：建议 160x160 像素，PNG 格式
- **位置**：根目录 `icon.png`
- **建议**：使用简洁的图标设计，如：
  - 📄 文档图标 + 箭头
  - ⬅️➡️ 左右箭头
  - 📖 翻页图标

### 2. preview.png
- **用途**：插件预览图，显示在插件市场详情页
- **尺寸**：建议 1200x675 像素（16:9），PNG 格式
- **位置**：根目录 `preview.png`
- **内容建议**：
  - 展示插件在文档底部的实际效果
  - 包含「上一页」、「下一页」按钮
  - 显示页码指示器（如：3 / 10）
  - 最好包含一些实际的文档内容作为背景

## 如何制作预览图

### 方法 1：截图（推荐）

1. 打开一个文档，滚动到底部
2. 确保导航按钮清晰可见
3. 使用截图工具（如 Windows 截图工具、Snipaste）
4. 裁剪为合适大小
5. 保存为 `preview.png`

### 方法 2：使用设计工具

1. 使用 Figma / Photoshop / GIMP 等工具
2. 创建 1200x675 的画布
3. 添加思源笔记界面的背景
4. 突出显示插件功能
5. 导出为 PNG
