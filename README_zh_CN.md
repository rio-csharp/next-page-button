# 文档导航按钮

[English](./README.md) | 中文

> 在文档底部添加「上一页」和「下一页」按钮，按文档树顺序浏览所有文档。

##  功能特性

- **顺序导航**：通过上一页/下一页按钮按文档树顺序浏览
- **位置指示**：显示当前位置（如：5 / 20）
- **实时更新**：自动响应文档树变化
- **多笔记本**：支持所有打开的笔记本
- **主题适配**：完美融入思源笔记界面
- **零配置**：开箱即用

##  安装

### 从插件市场安装（推荐）

1. 打开思源笔记
2. 进入 `设置`  `集市`  `插件`
3. 搜索 "Next Page Button" 或 "文档导航按钮"
4. 点击下载并启用

### 手动安装

1. 从 [Releases](https://github.com/rio-csharp/next-page-button/releases) 下载 `package.zip`
2. 解压到 `{workspace}/data/plugins/`
3. 重启思源笔记
4. 在 `设置`  `集市`  `已下载` 中启用

##  使用方法

安装后插件自动工作：

1. 打开任意文档
2. 滚动到文档底部
3. 使用导航按钮在文档间移动
4. 页码指示器显示当前位置

**按钮状态：**
-  **可用**：蓝色按钮可点击
-  **禁用**：灰色按钮表示已是第一篇/最后一篇

##  实时更新

导航会在以下情况自动更新：
- 创建或删除文档
- 移动或重命名文档
- 拖拽调整文档顺序
