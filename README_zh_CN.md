
# 下一页按钮 (Next Page Button)

[English](https://github.com/rio-csharp/next-page-button/blob/main/README.md) | 中文

> 像阅读电子书一样顺序浏览文档。根据文档树的层级结构，在页面间无缝切换。

## ✨ 功能特性

- **灵活布局**：支持 **底部固定** (经典) 或 **侧边悬浮** (现代) 两种布局模式，可在设置中自由切换。
- **智能导航**：严格遵循文档树顺序，支持笔记本隔离及拖拽重排序后的实时更新。
- **交互式 UI**：实时显示当前位置进度（如 `5 / 20`），完美适配各种主题。
- **无缝同步**：对文档的创建、删除、重命名或移动操作实现毫秒级响应。
- **移动端优化**：在 Android/iOS 上开启虚拟键盘时自动隐藏，防止遮挡。
- **多语言适配**：完整支持中英文，设置切换后 UI 立即生效，无需重启。

## 🚀 安装指南

1. 打开 **设置** -> **集市** -> **插件**。
2. 搜索 `Next Page Button`。
3. 点击 **下载** 并 **启用**。

## ⚙️ 插件设置

进入插件设置界面可自定义：
- **布局模式**：切换底部导航或侧边悬浮。
- **页边距**：调整底部布局的间距。
- **语言设置**：手动覆盖或随系统语言自动适配。

## 📱 移动端体验

针对小屏设备进行了专门优化：
- 编辑模式下悬浮按钮自动隐藏。
- 调用思源原生移动端 API，切换更流畅。
